# conixBot-Node
conixBot Node.js 版本

## 目前实现的功能

+ **#hi** 打招呼

    ![#hi](assets/hi.jpg)

+ **#say** 利用Google TTS 把语音转化为 语音发送到qq群里。
    ![#say](assets/say.jpg)

+ **#img** 输入一张图片的url 机器人自动把它发送到群里
    ![#img](assets/img.jpg)

+ **#upload** 输入文件直链与文件名 机器人自动把它上传到群文件
    ![#upload](assets/upload.jpg)

+ **#nslookup** 利用 ip-api.com 解析 域名/ip/url 的物理地址
    ![#nslookup](assets/nslookup.jpg)
